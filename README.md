# Inslider — Privacy Policy

Hosting for the Inslider app privacy policy.

- **main** branch → Google Play version (`index.html` / `privacy_policy.html`)
- **huawei** branch → Huawei AppGallery version (payments wording adjusted)

Enable GitHub Pages (Settings → Pages → Deploy from branch) and paste the
resulting URL into the store listing's Privacy policy field.
